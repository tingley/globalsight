Supported Data Formats

The following data formats are supported:

 

Excel2003

Excel2007

HTML

Illustrator

INDD (CS2)

INDD (CS3)

INDD (CS4)

INDD (CS5)

INDD (CS5.5)

InDesign Markup (IDML)

INX (CS2)

INX (CS3)

Java

JavaProperties

JavaProperties (HTML)

JavaProperties (Msg)

Javascript

JSP

MIF 9

Office2010 document

OpenOffice document

Passolo 2011

PlainText

Portable Object

PowerPoint2003

PowerPoint2007

Resource Compiler

RESX

RTF

Un-extracted

Word2003

Word2007

Xliff

XLZ

XML

 

* These format types require the purchase of document adaptor/converter.

* HTML can contain Javascript and CSS - the parser adjusts automatically.

* Passolo 2011 only supports files created by Passolo 2011 version. Because the files created by older version can not be used in original version any more after converted by Passolo 2011 version, GlobalSight will not process these files successfully with this format, instead, put them in a pending job with error in it.
