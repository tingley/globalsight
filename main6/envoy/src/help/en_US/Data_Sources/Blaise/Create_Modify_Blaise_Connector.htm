Create/Edit Blaise Connector

Summary:

Configure Blaise Connectors to view Blaise entries and create jobs.

Name:

The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional, description of current Blaise server.

URL:

The URL of the Blaise server.

User Name:

The user name to log onto Blaise server.

Password:

The password to log onto Blaise server.

Automatic Job Creation:

If the Blaise server turn on the automatic job creation feature.

Pull Time:

The schedule to create job with entries fetched from Blaise server automatically.

Combine by language:

If combine entries by language when creating job.

Minimum Word Count for Procedures:

The minimum word count for procedure type entries.

Default File Profile:

The file profile which is used in automatic job creation.

Attribute Group:

The name of attribute group which is used in automatic job creation.

User can set up different attributes for three kind of job types, ANY/HDU Workbook/ISheet.

Check Duration:

The schedule used to check if there has new entries in Blaise server, and the time unit is minute.

Buttons

 

Cancel

Cancel the creation/modification of the Blaise Connector.

Save

Save the added/modified Blaise Connector. When save, it will validate the URL/User Name/Password.
