Create Job

Summary:

Upload several files to create a job in the system.

Add Files Button

Press the button to select files or directoies to upload.

Recent Files Button

Press the button, a new window will open. Select files or directoies that had already been uploaded onto server, and press Add Files button, files will be added to file area for creating job.

Clear Profile Button

Clean all mapped file profiles of each file.

Job Name Text

Input the job name in the text. It is a required element.

Comment Text

Input some comments to describe the job. Comments must be less than 4000 characters.

Priority

The priority of the job. There are 5 levels of priorities.

Include as Job Support File

The attachment file type will be change to the "job support file".

Attachment -- Browse button

Press the button to select a file as attachment. If selecting several times, the last selected file will replace the former files.

Checkbox in front of "Target Locales"

Select/unselect all target locales.

Create Button

Create a job using the uploaded files.

Cancel Button

Clean all inputs and uploads. Don't create a job.

Note:

You can add common files and "zip, rar, 7z" compression file formats. When the file is 7z, your compressed file can support different compress method. But the rely jar cann't support all compress method(like ppmd), then the system sees the uncompressed file as a common file.
