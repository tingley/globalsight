MindTouch Connector Main Screen

Summary:

Configure MindTouch Connectors to view MindTouch pages and create jobs.

Table

Display all MindTouch Connectors that have been added to the system.

Buttons

 

Remove

Remove the selected MindTouch connector(s).

New...

Add a new MindTouch connector to the system (see Create/Edit MindTouch Connector).

Connect

Connect to selected MindTouch server.
