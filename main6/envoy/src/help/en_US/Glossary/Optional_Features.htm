Optional Features

Features marked optional may not be available in the system.  These features are available as separate add-on modules to the system.

 

Available Modules Include:

  • Security Package (SSL)

  • Individual Adaptors/Converters (Office, Quark, Frame, PDF)

  • Add/Delete Snippets

  • Job Costing

  • Reporting

  • Project Management (Calendaring)

  • Corpus TM

  • Vendor Management

  • Content Management

  • Aligner
