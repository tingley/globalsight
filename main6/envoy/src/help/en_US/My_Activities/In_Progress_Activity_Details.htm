In Progress Activity Details

Summary:

Work on an activity you have accepted.

Tabs

 

Detail

Refresh the detail page

Comments

View/Create/Edit comments for this activity (see Comments)

Work Offline

Download/upload offline files (see Upload or Download).

QA Checks

Manually run QA Checks and generate QA Reports (see Upload QA Report or Download QA Report).

Details

 

Job Name:

Name of the job, may be automatically generated by the system.

Job ID:

Numeric link for the job.  This is automatically generated by the system.

Activity:

Describes the type work to be performed.

Examples: Translation, Edit, Review

Project:

Name of the project.

Example: World Services R&D

Project Manager:

GlobalSight manager of the above project.

Source Word Count Total:

Total word count for all files in the job.

Priority:

A number from 1 to 5 indicating the urgency to localize this content.

Source Locale:

The source locale of the content in this activity.

Target Locale:

The target locale for this activity.

Due By:

The activity needs to be completed by the listed date.

Overdue:

Indicates if the activity's deadline has been missed.

Status:

Indicates whether the activity is still available, has already been accepted, or rejected.

Page/Hours (for Costing)

Enter the number of pages in the job or the number of hours you spent working on the job.  Click the Save button when you are done.

Primary Target Files

 

Target Page Name Search:

Filters pages whose page name contains the text in the adjacent blank text field.

(page name)

Click on a page name to see the content to localize.  If you have accepted the activity you can edit the content.  If you have not, the content is displayed in a read only mode.  Right click to choose the paragraph or the popup editor.

Search Text

Filters pages which contains a specific string. If click on the page to go to popup editor, corresponding occurrence is marked as yellow background.(The search text can not contain #,%,^,&,+,\\,\',\",<,>.).

Word Count

Lists the number of words for each page in this activity.  

This icon next to a file name indicates that it is an extracted file.

This icon next to a file name indicates an un-extracted file.

Buttons

 

Search (First)

Run Target Page Name Search.

Search (Second)

Run Target Page Text Search.

Download Source Files

Download all source files

Detailed Word Counts...

View word counts (see Detailed Word Counts)

Secondary Target Files

 

(page name)

Click on a page name to view the content to localize.

Buttons

 

Reject

Stop work on an activity.

This activity is now available for the original recipient(s) to work on

Task Completed

Move the activity on to the next step.

You cannot make any further changes to an activity you have finished

Update Leverage

Update leverage from in-progress jobs TM or/and re-leverage reference TMs which are specified by TM profile.

Leverage MT

Leverage MT in current activity. This button is only available for the workflow which has MT profile attached.

Export...

Available to Workflow Managers, Project Managers and Administrators only.  Opens the Export screen for the target locale in the current job.

Export & Download...

Export and download target files for this activity.you may need relevant permissions to enable this feature.

Save

 

Saves the number of pages or hours entered.  This button only appears if this activity has costs associated with the number of pages or hours.

 

 

As a project manager, you are unable to reject or finish activities that have not been assigned to you.
