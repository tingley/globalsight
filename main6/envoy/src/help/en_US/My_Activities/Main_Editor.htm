Main Editor (Popup)

Summary:

Localize content online

Note: If you have not accepted the activity, you will not be able to click on a segment to edit it.  If you are accessing this screen from My Jobs, you will see a Target Locale drop-down where you can select a target locale to view.

{segment}

Click any segment to edit (see Segment Editor).

You must accept an activity before you can edit anything.  If the segment editor does not launch after clicking a segment, close the main editor, accept the activity, then click a page name to re-open the main editor.  

 

In review mode, segments cannot be edited.  When comments are shown, clicking a segment will open the add/edit comments window for that segment.

List

Shows segments listed sequentially.

Text

Shows segments listed with non-localizable code, such as HTML.

Preview

Shows a preview of the page, segments are still editable. (Not available for all content types.)

Layout Modes

View Source and Target side-by-side.

View Target and Source above-and-below.

View Source or Target only (select the Source or Target radio button).

 

File Navigation

Move to the previous file in the activity.

Move to the next file in the activity.

Page Navigation

Move to the previous page in the activity.

Move to the next page in the activity.

Goto: Move to any page in the activity.

Unlock/Lock

Unlock: Unlocks all exact matches and allow them to be edited.

Lock: Re-locks unlocked exact matches.

If your Project Manager has not enabled the Unlock option, the Unlock/Lock button is not shown.

Show/Hide PTags

Display segments with/without PTags.

Show/Unmark Repeated

Show Repeated: Find all repetition segments to show to user. When show repeated/repetitions, one more column will appear next to segments. For "Repeated",its background is GREY,for repetition, its background is RED.

Unmark Repeated: Hide segments' repetition flags.

This is only availabe for edit role.

Auto-Propagate

Click to open Auto-Propagate dialog.

This is only availabe for edit role.

Progress

Opens a Review Window to highlight segments by status and jump to untranslated or fuzzy segments.

Show/Hide Comments

Opens/Closes the comment window.  When comments are shown, the comments icon appears next to segments with comments.  When the comments window is open:

  • Click any segment in the main editor to add/edit a comment for that segment.

  • Click the comments icon to add/edit a comment.

  • Click a comment in the comments window to jump the main editor to that segment.  

  • Double click a comment to edit or add to the comment.

You are only able to edit a segment comment you have added during this activity.

Support Files

Shows support files available for localization reference.

Termbases

Opens termbases associated with this project.

Snippet Editor

Opens the Snippet and Content Editor, which allows you to add and delete Snippets from target page.

Page Info

Includes troubleshooting information about a page that may assist GlobalSight Customer Support.

Options

Opens a dialog window that lets you change the default link colors of the Editor.

Preview (link appears next to database records)

Click this link to open a read-only preview of the Source and Target database records.

This link only appears if you are localizing database records.

Filter (Dropdown)

Filter the segments by categories.

Target Locale (Dropdown)

This dropdown only appears when the page is opened by the project manager from the Job Details screen.  Select one of the target locales in the Job to display the current state of the page for this locale.

Close

Closes the main editor.

Your work is saved as you edit segments in the segment editor, no additional save is necessary.
