Update Leverage Options

Summary:

Select to update leverage from in progress jobs TM or/and to re-leverage reference TMs.

Update Leverage Options

 

Update from Jobs

Checked: Will update from specified jobs' in progress TM data. User must select jobs to update from.

Unchecked:Will not leverage from any in-progress jobs' translations.

In-Progress TM Penalty

TM penalty for in progress translations from specified jobs. Default 0. If the penalty is 0, any 100% matches will be populated into target TUV.

Re-apply Reference TMs

Checked: Re-leverage reference TMs just as that while creating jobs. If there are new matches from gold TM, store them into DB too. 100% matches can be populated into target TUV just as that while creating jobs.

Unchecked: Not re-leverage reference TMs.

Re-try MT

Checked: Will re-try MT for untranslated segments.

Unchecked: Not re-try MT.

Buttons

 

Cancel

Cancel update leverage and turn back to task details page.

Update

Update leverage and turn back to task details page.
