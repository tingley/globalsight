Job SourceFiles

Summary

View detailed information about a job.

Job ID:

The job ID assigned by the system.

Job Name:

This is the name given to the job by the initiator or automatically generated by the system.

Initiator:

The name of the user who created the job, or the project manager if the job was automatically created.

Source Locale:

The source locale of the content in this job.

Project:

Name of the project.

Example: World Services R&D

Localization Profile:

The localization profile used to create this job.

This determines the workflows and their activities for this job.

Date Created:

Date and time that the job was created.

Pages:

The number of pages in the job. This field is part of the Job Costing feature.  Click to edit the number of pages in the job.

Priority:

The priority assigned to this job.

Primary Target Files

 

Source Page Name Search:

Filters pages whose page name contains the text in the adjacent blank text field.

(page name)

Click on a file name to open the page in the Main Editor (Popup).
Note: The file may be in the process of being edited by another user.  When working online, each segment is saved individually.  Subsequent changes to a segment will overwrite previous ones.  Special care should be taken when editing files online that are currently being edited offline by a user.  When the user uploads the file, all online changes made after the user downloaded the file to work on are lost.

This icon next to a file name indicates that it is an extracted file.

This icon next to a file name indicates an un-extracted file.

Source Word Count Total:

The total word count for all of the source pages in the job.  Click Edit to change the wordcount for the job.

Search Text:

Filters pages which contains a specific string. If click on the page to go to popup editor, corresponding occurrence is marked as yellow background.(The search text can not contain #,%,^,&,+,\\,\',\",<,>.).
