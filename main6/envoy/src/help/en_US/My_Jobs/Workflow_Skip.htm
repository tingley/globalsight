Skip Activities

Summary:

Force the workflow(s) to the activity you selected. The current activity will be skipped after this operation.
Only the workflows that are to be dispatched and dispatched are allowed and necessary to be done this operation.

Items

 

Target Locale:

The target locale of the workflow.

Activities in the Default Path:

The advanced activities in the default path of the workflow.

Buttons

 

Cancel

Return to Job Details, without making changes.

Save

Return to Job Details, and save the changes.
