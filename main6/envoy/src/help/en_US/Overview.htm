Overview - Administrator

Getting Started

Setup Guides for:

FileSystem and Database.

Each of these guides steps you through the process of configuring GlobalSight for the content type listed.

Menus

Menus for quickly accessing product functionality.

Setup and Data Sources contain links to all of the functionality listed below.

Guides contains links to all available system guides for configuring GlobalSight.

My Jobs and My Activities contain links to display all jobs or activities in a particular state.  Once a search has been performed, the menu contains a link to the most recent search.  These menus also contain links to the three most recently accessed jobs and activities.

Reports contains links to reports and reporting functionality.

Vendor Management contains links to all Vendor Management features.

Content Manager contains a link to access the content management console.

 

Setup

Locale Pairs

Identify source-target locales for system-wide use.

Activity Types

Create/Edit activities for users to perform as part of the localization process.

Currency*

Specify the currency conversion rate from various currencies to the system currency.

Rates*

Determine the per word costs for activities in locale pairs.

Calendars/Holidays*

Defining working days, working hours, and holidays.

Users

Create/Edit user information and access to the system.

Translation Memory

Create empty TMs in which to store translations.

Translation Memory Profiles

Establish parameters for how translation memory should be used within the system.

Machine Translation Profiles Profiles

Establish parameters for how machine translation should be used within the system.

Terminology

Maintain terminology databases to be used as references during the localization process.

Projects

Create/Edit projects to match content localization groups.

Workflows

Create/Edit workflows for routing translation jobs through the system.

Localization Profiles

Create/Edit profiles for completing localization work.

Support Files

Upload locale-pair specific glossaries for localization participants.

Snippet Import*

Import snippets in GlobalSight to populate the Snippet Library.

System Parameters

Change several system-wide parameters (incl. fuzzy matching threshold, web server admin email).

Logs

GlobalSight logs to assist in resolving system issues.

 

Data Sources

File Profile

Set the encoding, file format and localization profile to use for importing content.

File Extensions

Create new file extensions for supported file formats.

XML Rules

Create sets of rules to determine the XML content to be localized within XML schemas.

Import

Manually import files into GlobalSight for localization.

Database Connections*

Specify connection strings and drivers for reading/writing to databases.

Database Profiles*

Specify read/preview/write SQL statements for localizable information in database tables.

Database Import Profiles*

Define pending records tables to monitor and set subsequent job size parameters (see also Pending Records Table).

DB Preview*

 

Create instructions, written in XML, detailing the path to a template page for displaying source and target data dynamically during preview.

Export Locations

Create/Edit directories where you can export data to.

Vignette Import*

Connect to a Vignette installation to import data for localization.

Corpus Aligner*

Create, download and upload alignment work.

RSS Reader*

Subscribe, unsubscribe and translate blog content.

 

Common

My Jobs

Details the status of all jobs in the system (Administrator) or all jobs created for your project (Project Manager).

My Activities

Shows all activities assigned to you.

Reports

Shows reports on system usage.

Vendor Management*

Create/Edit vendor user information and access to the system.   (Only available to users with Vendor Management permission.)

Content Management*

Displays the Content Management console.  You must have a valid CMS username and password for this function to work.

* These are optional features and may not appear in the list of menu options.  These features are available as separate add-on modules to the system.
