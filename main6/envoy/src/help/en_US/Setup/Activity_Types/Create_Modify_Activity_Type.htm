Create/Edit Activity Type

Summary:

This activity type can be added as a part of workflow templates in localization profiles.

Activity:

This name should describe the type of work to be done, e.g. Translation.  This value cannot be edited after the object has been saved.  The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Note: Although the entry box allows for longer names, activity type names are automatically shortened to 29 characters.

Description:

This description is not seen by workflow participants.

Auto Complete Activity:

Unchecked by default. If checked and activity has been accepted, it will be completed automatically after the configured time.

After Job Creation:

Activity complete time is calculated from job is created.

After Job Dispatch:

Activity complete time is calculated from job is dispatched.

After Activity Start:

Activity complete time is calculated from activity is accepted.

d/h/m:

d(day), h(hour), m(minute).

Review Only:

Checked: Participants in this activity will have read only access to the content in the online and offline editors.

Unchecked: Participants in this activity will be able to edit content normally through the online and offline editors.

QA Checks:

Checked: Indicate that this activity is a QA-based activity which should run the QA checks automatically and create a QA report if the job actually has an active QA filter.

Unchecked: Indicate that this activity is not a QA-based activity and no QA checks will be done.

DITA QA Checks:

Unchecked default. If checked, when workflow comes to this activity, it will auto-run DITA QA Checks and generate DITA report before it is accepted.

Buttons

 

Cancel

Cancels the activity type creation/edit.

Save

Saves the new/changed activity type.
