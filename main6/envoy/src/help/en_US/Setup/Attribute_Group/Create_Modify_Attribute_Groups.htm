Create/Edit Attribute Group

Summary:

Create or edit an attribute group.

Basic information

 

Name:

For displaying in GlobalSight. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional, viewable by administrators and project managers in GlobalSight.

Selected Attributes:

All selected attributes.

Available Attributes:

All available attributes.

Buttons

 

Cancel

Cancels creating or editing attribute group.

Save

Saves the attribute group.
