Create Company

Summary:

Add a company to the system.

Name

Company Name.

Description

The description of the company.

Email Address

The From Address of email in the company.

Session Timeout

The User Session Timeout (minutes) of the company.

Enable IP Filter:

Default is enable. If checked, GlobalSight will check the IP of the remote Web Services vistiors according to the Remote IP Filter( Setup->Remote IP Filter for Webservices) list.

Enable TM Access Control:

Default is disable. If checked, GlobalSight will start TM access control for users.

Enable Terminology Access Control:

Default is disable. If checked, GlobalSight will start Terminology access control for users.

Use Separate Tables per Job:

Default is unchecked. If checked, every job will has its own TU/TUV/LeverageMatch tables for performance purpose.

Enable QA Checks:

Default is unchecked. If checked, the enabling of the QA checks will be allowed for this company.

Enable Trisoft (LiveContent Architect) DITA Checks:

Default is unchecked. If checked, it will enable DITA QA Checks feature in this company.

Enable Workflow State Posts:

Default is unchecked. If checked, it will enable Workflow State Posts feature in this company.

In Context Review:

Options to control In Context Review tool function in this company. Please enable "In Context Review Tool" first when installing GlobalSight.

      InDesign:

Default is unchecked. If checked, it will enable In Context Review Tool for InDesign (indd, idml) files in this company. Please enable "In Context Review Tool" and set conversion directory for InDesign first when installing GlobalSight.

      Office 2010:

Default is unchecked. If checked, it will enable In Context Review Tool for Office (docx, pptx, xlsx) files in this company. Please enable "In Context Review Tool" and set conversion directory for Office first when installing GlobalSight.

      XML:

Default is unchecked. If checked, it will enable In Context Review Tool for XML files in this company. Please enable "In Context Review Tool" first when installing GlobalSight.

Define available segment comment categories for this company used on the job segment comment page:

Available Categories: Available categories for company.

Added Categories: The company categories.

Add A New Category: Creates new category for company.

Define available reviewer scorecard categories for this company used on the reviewer scorecard page:

Available Categories: Available scorecard categories for company.

Added Categories: The company scorecard categories.

Add A New Category: Creates new scorecard category for company.

Define available Quality Assessment categories for this company used on the Post Review QA Report:

Available Categories: Available quality categories for company.

Added Categories: The company quality categories.

Add A New Category: Creates new quality category for company.

Define available Suitability for Market categories for this company used on the Post Review QA Report:

Available Categories: Available market categories for company.

Added Categories: The company market categories.

Add A New Category: Creates new market category for company.

Define available DQF Fluency values for the Reviewers DQF scorecard page :

Available Categories: Available DQF(Dynamic Quality Framework) Fluency categories for company.

Added Categories: The company DQF Fluency categories.

Add A New Category: Creates new DQF Fluency category for company.

Define available DQF Adequacy values for the Reviewers DQF scorecard page:

Available Categories: Available DQF Adequacy categories for company.

Added Categories: The company DQF Adequacy categories.

Add A New Category: Creates new DQF Adequacy category for company.

Define available DQF segment comment Severity values:

Available Categories: Available DQF segment comment Severity categories for company.

Added Categories: The company DQF segment comment Severity categories.

Add A New Category: Creates new DQF segment comment Severity category for company.

Define default value for the DQF Fluency and Adequacy fields

Fluency -- Set default value of Fluency for the company

Adequacy -- Set default value of Adequacy for the company

Add A New Category: Creates new DQF segment comment Severity category for company.

Buttons

 

Cancel

Cancels the company creation.

Save

Saves the new company.
