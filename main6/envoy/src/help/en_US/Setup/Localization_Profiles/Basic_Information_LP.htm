Localization Profile Basic Information

Summary:

Set Job Size and Dispatch Criteria, TM Options, Project (Manager), and Priority.

Name:  

For display in GlobalSight. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional (for display to project managers and administrators).

Optional Script:

Optional (can be used for additional job notification).

Translation Memory Profiles:

Select the TM Profile to be used with this localization profile.  

Project:

Select the project – the project manager for this project and the workflow manager are the only users who can modify workflows once they are created.

Priority:

Assigns a priority (1-5) to activities associated with workflows created by this localization profile; users completing these activities see this priority.

Source Locale:

Select the locale of source content.

Use Translation Memory:

Allow Edit of ICE and 100% Locked Segments - New content is leveraged against TM; Exact Matches (locked) can be changed (unlocked).

Allow Edit of ICE Locked Segments - New content is leveraged against Exact Matches (locked) can be changed (unlocked).

Allow Edit of 100% Locked Segments - New content is leveraged against TM; (locked) can be changed (unlocked).

No - New content is not leveraged against TM, although finished translations are still written to TM.

Use MT Engine:

Automatic – Automated use MT engine for workflows to which have MT profiles attached during job creation. Default.

Manual – Do not use MT engine during job creation. After job is created, user can leverage MT manually.

Workflow Dispatch:

Automatic – Once an import is complete, it moves from Pending to In Progress, and the first participant is notified to begin.

Manual – Once an import is complete, the job is placed in Ready.  From Ready, the project manager can decide whether to discard the job or move it to In Progress.

Attach Workflow:  

Attached workflows to this localization profile. Select a workflow by clicking the ComboBox in the right of every target locale.

MT Profiles:  

Attached MT Profile to this localization profile. Select a MT Profile by clicking the ComboBox in the right of every workflow.

Buttons

 

Cancel

Cancels localization profile creation/edit.

Save

Save the localization profile.
