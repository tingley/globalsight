Create/Edit Perplexity Service

Summary:

The Perplexity Service determines how to connect perplexity service.

Name:

The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional, viewable by administrators and project managers in GlobalSight.

User Name:

The user name that used to connect perplexity.

Password:

The password that used to connect perplexity.

URL:

The URL that used to connect perplexity.

Buttons

 

Cancel

Cancels the creation/modification of the Perplexity Service.

Save

Saves the added/modified Perplexity Service.
