System Restart

Summary:

Schedule and provide notice for a system restart.

Hours:

Specify the number of hours until the system restart.

Minutes:

Specify the number of minutes until the system restart.

Message:

Specify the message that will appear to users until the system restart occurs.  The time remaining until restart will always display for users.

Buttons

 

Submit

Submits the system restart request.  The system will restart in the number of hours and minutes specified from the time of the request.

Home

Returns to the main Home screen.

If a restart has been scheduled...

 

Change delay until restart

Select this option, enter new values under hours and minutes and click Submit.

Cancel the existing restart request.

Select this option, then click Submit.
