System Shutdown

Summary:

Schedule and provide notice for a system shutdown.

Hours:

Specify the number of hours until the system shutdown.

Minutes:

Specify the number of minutes until the system shutdown.

Message:

Specify the message that will appear to users until the system shutdown occurs.  The time remaining until shutdown will always display for users.

Buttons

 

Submit

Submits the system shutdown request.  The system will shut down in the number of hours and minutes specified from the time of the request.

Home

Returns to the main Home screen.

If a shutdown has been scheduled...

 

Change delay until shutdown

Select this option, enter new values under hours and minutes and click Submit.

Cancel the existing shutdown request.

Select this option, then click Submit.
