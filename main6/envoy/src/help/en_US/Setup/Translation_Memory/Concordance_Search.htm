TM Search

Summary:

Lists matching strings found in the TM.

Search Type

There are two different search types. FullText search is based on Lucene(Points word search). Exact/Fuzzy match based on leverage, leverage option from TM Profile.

Search Text

Enter a string in the source locale to find in the TM.  All words in the string are used to find matches.  For example, searching for the string 'Acme brand' might result in the following matches: "The Acme Brand was founded in 1970." and "Acme is a brand recognized in more than 74 countries.".

 

Enter a string in the source locale to find in the TMs.  The entire string is used to find matches in the TM, and is compared to entire segments.

Translation Memory

Select TMs in which to search.  Depending on your system configuration, only the TMs used in the project for this activity are listed, or all TMs are listed.

TM Profile

Select the TM Profile to use for executing the search.  The TM Profile will specify the fuzzy match penalties for white-space, case, code, type and multiple exact matches.

TU ID

Filter by TU id.

Attribute Name

Filter by attribute name.

Attribute Value

Filter by attribute value.

Creation User

Filter by creation user.

Modification User

Filter by modification user.

Job ID

Filter by job id.

Created on

Filter by creation date.

Modified on

Filter by modification date.

Last Usage Date

Filter by last usage date.

 

Buttons

 

Search

Initiates the search.

Add

Invoke a page to add a new entry.(Add Entries permission)

Delete

Delete selected entries.(Delete Entries permission)

Source(link)

User could click source link to invoke edit entry page to edit entry.(Edit entries permission)

Advaced(link)

In "Advanced" search mode, user can search in source or target, can search by creation date and modify date, can search with replace text, can search with tu id, string id, create user, modify user, attribute name and attribute value. If use replace with function, need click apply button to save.

 

Search Results Table

 

Percentage

The percentage of the fuzzy match retrieved (Exact/Fuzzy Match Search Only).

Source Locale

Lists the matching source segment.  The string used for the search is highlighted.

Target Locale

Lists the corresponding target segment.

SID

Lists the corresponding sid.

TM Name

Lists the corresponding tm name.
