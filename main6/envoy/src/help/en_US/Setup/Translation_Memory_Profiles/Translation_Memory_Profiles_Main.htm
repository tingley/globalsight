Translation Memory Profiles Main Screen

Summary:

Establish parameters for how translation memory should be used within the system.

Table

Displays all translation memory profiles that have been added to the system.  Click a column header to sort the table based on data in that column; click a second time to reverse the sort order.  Select a translation memory profile by clicking the checkbox next to the translation memory. Click the "name" link to edit TM profile. Input string for "name", "storage tm", "company" and click "enter" to filter data.

Buttons

 

New...

Add a new TM profile to the system (see Create/Modify TM Profile).

Remove...

Remove the selected TM profile.
