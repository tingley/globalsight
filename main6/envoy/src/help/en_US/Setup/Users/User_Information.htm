User - Basic Information

Summary:

Create or modify a user in GlobalSight.

Username:

User login.  The login may only contain letters, numbers, the underscore "_", the hyphen "-", the at symbol or dot '.'.

First Name:

For display purposes.

Last Name:

For display purposes.

Password:

No restrictions on length or content.

If current company enables 'Enable Strong Password' option, then the password must use a strong one.

Strong password is at least 8 characters long. And it should contain 3 of below items at least.

  • Digits: Integer values between 0 and 9.
  • Uppercase characters: Like A,B,C......
  • Lowercase characters: Like a,b,c......
  • Special characters: Like %,$,^,~......

Repeat Password:

Re-enter the password.

Title:

For display purposes.

Web Services Password:

The full name is Web Services Security (WS-Security, WSS) password credential, it is used to login GlobalSight Axis2 Web Services, like COTIApi.

Company Name:

Select a company from the dropdown list, or enter the name of a new company in the text box provided.

 

 

Buttons (New)

 

Cancel

Cancel the user account creation.

Next

Enter the user's contact information (see Contact Info).

Buttons (Edit)

 

Cancel

Cancel the user account information edits.

Roles...

Edits the user's roles (see New Roles, Edit Roles).

Default Roles...

Edits the user's default roles (see New Roles, Edit Roles).

Calendar... (Optional)

Edit the user's calendar (see Edit Calendar).

Contact Info...

Edit the user's contact information (see Contact Info).

Projects...

Specify the projects to which the user is assigned (see Projects).

Security...

Set access to the fields describing this user (see Field Level Access).

Permissions...

Edit the user's permission (see Permission Groups).

Save

Saves edits to the user's account information.
