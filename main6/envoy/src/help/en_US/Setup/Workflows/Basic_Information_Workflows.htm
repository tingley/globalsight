Enter/Edit Basic Information for New Workflow

Summary:

Set Name, Description, Project Manager, Locale Pair for the new workflow.

Name:  

For display in GlobalSight. The name may only contain letters, numbers, the underscore "_" or the hyphen "-".

Description:

Optional (for display to project managers and administrators).

Project:

Select the project in which this workflow is used.  Only users assigned to this project may perform work for this workflow.

Notify Project Manager via email

Checked: The project manager is notified via email about workflow progress (activity acceptance, activity completion, etc.).

Unchecked:  the project manager is not notified via email about workflow progress.
This value cannot be changed once the object has been saved.

Workflow Manager:

Select one or more workflow managers (optional).  The workflow managers are not responsible for the entire job, but only designated workflow(s) within the job.

Locale Pair:

The locale pair that this workflow can be used for.

Target Encoding:

This encoding is shown as the default for manual exports of primary target files or secondary target files.  For automatic export of primary or secondary target files, this is the encoding that is used.  Note that certain content types (Microsoft Office) override this selection (to UTF-8).

Leverage From:

Choose locale pairs to leverage TM from when importing a file. You can choose from all locales that share the same language.

Scorecard Show Type:

Scorecard show types for workflow. There are several chioses can be selected,

  • Not Showing -- Don't show Scorecard
  • Scorecard Optional -- Scorecard is optional for the workflow
  • Scorecard Required -- Scorecard is required for the workflow
  • DQF + Scorecard Optional -- DQF and Scorecard is optional for the workflow
  • DQF + Scorecard Required -- DQF and Scorecard is Required for the workflow
  • DQF Optional -- DQF is optional for the workflow
  • DQF Required -- DQF is Required for the workflow

Buttons

 

Cancel

Cancels workflow creation/edit.

Next

Moves to the next screen of workflow creation/edit.
