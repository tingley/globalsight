Crear/modificar conexión

Sólo podrá ver estas pantallas si cuenta con integración de bases de datos.

 

Resumen:

Permite especificar una cadena de conexión y un controlador para leer y grabar en una base de datos.

Nombre:

Para visualización en System4.

Descripción:

Optativa, los administradores pueden ver esta descripción en System4.

Controlador de la base de datos:

Ruta completa de acceso a la clase Java al controlador de la base de datos que se emplea para comunicarse con su base de datos.

Cadena de conexión:

Utilice la definición normal de Java para la conexión a la base de datos.

<tipo_controlador>:<tipo_base_datos>:<tipo_conexión>:@<servidor.nombredominio.dom

inio>:<NúmPuerto>:<NombreProcesoInstanciaBaseDatos>

NOTA: El nombre de usuario y la contraseña no se incluyen en esta cadena. Se introducen en los campos "Nombre de usuario" y "Contraseña" que aparecen más abajo.

Nombre de usuario:

Nombre de este usuario de la base de datos.

Contraseña:

La contraseña del usuario cuyo nombre se ha introducido.
