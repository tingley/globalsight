TRADOS offers state-of-the-art software that allows you to distribute terminology off-line, e.g. on CD ROM, or on-line, e.g. on an Intranet or on the Web.

From this page you can freely download or access a number of sample dictionaries that allow you to test our terminology distribution software. 

MultiTerm Dictionary

MultiTerm Dictionary is a terminology lookup software that allows you to distribute and publish electronic dictionaries, for example on CD ROM. Below you find a list of freely available sample dictionaries. To look up the dictionaries listed below, you first need to download the MultiTerm Dictionary Lookup module, which can be downloaded for free (approx. 1.7 MB).  For more information on MultiTerm Dictionary, click here...

  • MultiTerm Dictionary Lookup Module (approx. 1.7 MB). To use the sample dictionaries listed below you need to have the MultiTerm Dictionary Lookup Module installed on your system. Download the dictionary of your choice, unzip all files contained in the ZIP archive into a folder on your local hard disk, and then open the file with the extension MD (e.g. euterpe.md) in MultiTerm Dictionary. 

 

  • EUTERPE (23 MB): The complete terminology collection of the European Parliament, contains over 195,000 entries in 11 languages. 
    Download  Greek  fonts (100 KB): These fonts must be installed to ensure the dictionary runs correctly. If these fonts are not made available to the program, your work may be interrupted by error messages.
  • BEUMER Dictionary (1.2 MB): Sample Dictionary containing approx. 100 engineering terms. This dictionary is courtesy of BERNHARD BEUMER MASCHINENFABRIK KG. 
    Download  Russian  fonts     (100 KB): These fonts must be installed to ensure the dictionary runs correctly. If these fonts are not made available to the program, your work may be interrupted by error messages.
  • OXFORD DUDEN (290 KB): Sample Chapter 'N' of the English-German dictionary Duden Oxford. 
  • IMF Terminology (39 KB): Sample Chapter 'A' of the English-French-Spanish dictionary by the International Monetary Fund. 
  • ASCOM Dictionary (450 KB): This sample dictionary (English, German, French, Italian, and Spanish) is courtesy of ASCOM in Bern.
  • DIN (320 KB): This sample dictionary is courtesy of DIN Deutsches Institut für Normung e.V. in Berlin.

 

MultiTerm Web Interface 

MultiTerm Web Interface was designed to distribute terminology on-line, e.g. on a company Intranet or on the Web. All you need to access the on-line dictionaries listed below is an Internet browser. For more information on MultiTerm Web Interface, click here...

  • EUTERPE: This dictionary was created by the European Parliament. It now contains over 150,000 entries in the 12 official languages of the European Community.
  • BEUMER: This sample database was compiled by Bernhard Beumer Maschinenfabrik KG, a machine manufacturer from Beckum, Germany.
  • HAUNI: This sample dictionary was created by the Hamburg-based machine manufacturer HAUNI and contains some 150 entries in up to five languages.
  • UPU: This sample dictionary was created by the UNIVERSAL POSTAL UNION (UPU) and contains entries in up to five languages.
